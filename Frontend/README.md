
  # App Builder

  This is a code bundle for App Builder. The original project is available at https://www.figma.com/design/D7tXGWxXf1ww6J0g5vLFhF/App-Builder.

  ## Running the code
  Run `npm i` to install the dependencies.
  Run `npm run dev` to start the development server.

## 실행
npm run dev    # 개발
npm run build  # 컴파일
npm run start  # 실행
